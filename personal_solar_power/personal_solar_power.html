This prototype allows selecting options from the dropdown menus
which are then logged on the console. Once all calculation formulas
are implemented these user selected options will be used as input values.
Also in this prototype are the plotted locations of all weather stations.
These are currently not interactive, but ultimately will get highlighted
on hover, show a tooltip and be clickable. This will then change the location
in the dropdown menu and update the weather data, there by all visualizations
and resulting numbers. Some weather stations are missing data (see the json
logged on the console) and will therefore not be suitable for some calculations.
